Review Game is a gamified web application for code review. Users find issues in code, perform reviews, and compete for scores.

## Overview

Review Game provides review evaluation based on actual code. The system is built with React Router and runs on Cloudflare Workers. User reviews are evaluated using the Gemini API.

## Key Features

- User code review input
- AI-powered review evaluation
- Result image generation and storage
- Result sharing via X integration

## Tech Stack

- Frontend: React Router
- AI: Gemini API
- Hosting: Cloudflare Workers
- Image Storage: Cloudflare R2

DROwl is an open-source DevRel (Developer Relations) support tool. It serves as an information collection platform and enables comparison with other services.

## Overview

DROwl runs on Cloudflare Workers and uses Hono as the application framework. It collects information through various APIs and performs analysis and report generation.

## Key Features

- DevRel activity data collection
- Account integration
- Report generation

## Tech Stack

- Frontend: React Router
- Backend: Hono (Cloudflare Workers)
- Database: PostgreSQL
- File Storage: S3 API

MOONGIFT is a blog that introduces open-source software. It ceased updates on July 16, 2021.

## Overview

Since its establishment in 2000, MOONGIFT has been providing daily information about open-source software and free software. Supported by many readers, it has been operated as a technical blog for a long time.

## Main Content

- Articles introducing open-source software
- Software usage guides and reviews

## Tech Stack

The technology stack during operation was as follows:

- Frontend: Ruby on Rails
- Backend: WordPress
- Database: MySQL

It is now archived and hosted as a static site with the following tech stack:

- Hugo
- Firebase Hosting

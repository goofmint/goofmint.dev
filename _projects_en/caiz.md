Caiz.dev is an online community service. Based on NodeBB, anyone can freely create their own community.

## Overview

This project is built on NodeBB with customized themes and plugins. Users can create a free account and launch their own community.

## Key Features

- Hosting multiple communities on a single NodeBB instance
- AI moderation
- AI-powered thread summaries
- AI-powered translation in 20 languages
- MCP server

## Tech Stack

- Frontend: NodeBB (Node.js, Express, Socket.io)
- AI: OpenAI API / Gemini API

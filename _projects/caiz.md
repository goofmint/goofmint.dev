Caiz.devは、オンラインコミュニティサービスです。NodeBBをベースとしており、誰でも自由にコミュニティを作成できます。

## 概要

本プロジェクトはNodeBBをベースとして、テーマやプラグインをカスタマイズすることで実現しています。ユーザーは無料でアカウントを作成し、自分のコミュニティを立ち上げることができます。

## 主な機能

- 1つのNodeBBインスタンスで複数のコミュニティをホスティング
- AIモデレーション
- AIによるスレッドサマリー
- AIによる20言語翻訳
- MCPサーバー

## 技術スタック

- フロントエンド： NodeBB (Node.js, Express, Socket.io)
- AI： OpenAI API / Gemini API

iOS・Android向けクロスプラットフォームモバイルアプリケーション。

## 概要

React Nativeを使用して構築されたこのアプリは、両モバイルプラットフォームでシームレスな体験を提供します。

## 主な機能

- クロスプラットフォーム対応
- オフラインサポート
- プッシュ通知
- クラウド同期

## 技術スタック

- フレームワーク: React Native
- 状態管理: Redux
- バックエンド: Firebase
- テスト: Jest

## 学んだこと

モバイル開発、状態管理、クラウドサービスとの連携について経験を積みました。
